[ NEURAL SCANNER v1.0 ]

Neural Scanner is a high-performance, aesthetically driven Email and SMS Spam Detection system. Designed with a custom-styled Cyberpunk terminal interface, it utilizes Machine Learning to intercept and analyze data streams for potential threats.


📊 Performance Metrics

System Accuracy: 95.46%

Algorithm: Multinomial Naive Bayes (MNB-V1)


🤝 Development Philosophy: Human-AI Collaboration

This project represents a strategic partnership between human creativity and artificial intelligence:

The Architect: Shaheer Ahmed (Logic design, system architecture, and final refinement).

AI Co-Pilot: Assisted in UI/UX styling, CSS animations, and code modularity.


⚡ Key Features

Neural Analysis Core: Powered by MNB for high-accuracy text classification.

Futuristic UI/UX: Custom CSS-injected Streamlit interface with CRT scan lines and animated binary headers.

Interactive System Radar: Visual radar component that alerts the user with "Threat Dots" when a spam signal is detected.

Dynamic Geo-Location: Integrated Pakistan-based coordinates (PK-IS) mapping for source signal simulation.

Audio Alerts: Real-time feedback with "Secure" and "Threat" signals upon scan completion.

Mobile Optimized: Fully responsive containers that adapt to mobile screens.


🛠️ Tech Stack

Language: Python

Framework: Streamlit (UI/UX Customization)

ML Engine: Scikit-learn

Data Processing: Pandas, Counter

Styling: Custom CSS (Fira Code, Neon Accents, Keyframe Animations)


🚀 Usage

Initialize System: Load the trained spam_model.pkl and words_list.pkl.

Paste Data Stream: Input any suspicious email or SMS text into the SIGNAL_ANALYSIS terminal.

Execute Scan: Run the decryption process to see if the signal is SAFE or a SPAM THREAT.

Note on Dataset: The emails.zip file contains the raw dataset in compressed format due to size limitations. Extract it if you wish to inspect the training data.
